# REST Robotics — Arduino Code Repository

**Roseville High School | REST Robotics Team**

This repository contains all Arduino code, tools, and shared libraries for the three REST Robotics competition teams.

---

## Teams

| Team | Competition Type | Folder |
|------|-----------------|--------|
| RC (Mini IVD) | Radio-controlled vehicle design | [`rc-team/`](./rc-team/) |
| V2X | Autonomous vehicle-to-anything | [`v2x-team/`](./v2x-team/) |
| Autonomous | Servo/sensor autonomous robot | [`autonomous-team/`](./autonomous-team/) |

---

## How Each Year Is Organized

Every team folder follows the same pattern inside each school year:

- **`milestones/alpha/`** — Code submitted for Milestone Alpha
- **`milestones/bravo/`** — Code submitted for Milestone Bravo
- **`milestones/charlie/`** — Code submitted for Milestone Charlie
- **`competition/`** — Final polished code used at the in-person event

## Getting Started

1. Install the [Arduino IDE](https://www.arduino.cc/en/software)
2. Clone this repo: `git clone https://github.com/YOUR-USERNAME/rest-robotics.git`
3. Open any `.ino` file in the Arduino IDE
4. See [`docs/hardware-setup.md`](./docs/hardware-setup.md) for wiring guides
